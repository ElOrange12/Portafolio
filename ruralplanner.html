Rural Planner

La herramienta definitiva para organizar viajes con amigos

Sobre el Proyecto

Rural Planner es una aplicación web colaborativa desarrollada junto a mi compañera Ruth. Nace para solucionar el clásico caos de los grupos de WhatsApp cuando se intenta organizar una escapada rural: decidir la casa, cuadrar fechas, calcular quién debe dinero a quién y organizar las comidas.

El proyecto simula un entorno real con un sistema de autenticación seguro, diferentes roles de usuario (Administrador y Usuario estándar) y una base de datos relacional que gestiona desde las votaciones de las casas hasta una calculadora dinámica de transporte y presupuestos en tiempo real.

Funcionalidades Destacadas

  • Módulos de Decisión: Sistemas de votación interactivos para elegir la casa rural y cuadrar fechas en un calendario dinámico.
  • Calculadora Financiera: División automática de gastos (compra a escote, peajes, gasolina, billetes) calculando exactamente a cuánto toca cada asistente.
  • Gestión de Archivos: Conversión nativa de imágenes a Base64 en el cliente (JavaScript) para subida de fotos en las propuestas de actividades y casas.
  • Exportación PDF: Generación automática de informes detallados o compactos del plan final listos para compartir o imprimir usando la API nativa del navegador.
  • Panel de Administración: Dashboard exclusivo para el rol "admin" que permite moderar usuarios, purgar casas, actividades y gestionar asistentes.
