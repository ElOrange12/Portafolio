AdminViews

Gestión de Películas y Series

Sobre el Proyecto

AdminViews es una aplicación web completa, desarrollada de forma colaborativa junto a mi compañero Valentín. El objetivo principal fue crear una herramienta práctica para resolver un problema común: el caos al intentar recordar qué películas o series tienes pendientes de ver o cuáles ya has terminado.

Este proyecto simula un entorno de producción real, implementando un sistema de autenticación seguro (Login/Registro), gestión de sesiones y una base de datos relacional robusta.

Características Principales

  • Gestión de Usuarios: Sistema de Registro e Inicio de Sesión seguro con encriptación de contraseñas.
  • CRUD Completo: Permite Crear, Leer, Actualizar y Borrar registros de películas y series de tu lista personal.
  • Filtrado por Estado: Organización automática en listas de "Pendientes", "Vistas" o "Favoritas".
  • Interfaz Responsive: Diseño adaptable que funciona tanto en escritorio como en dispositivos móviles.
  • Despliegue Real: La aplicación está alojada en un servidor web y es accesible públicamente.
