Duelo de Dragones

Simulador de Combate en Python

Sobre el Proyecto

Duelo de Dragones es un ejercicio práctico de programación estructurada en Python, diseñado para aplicar conceptos fundamentales como la creación de funciones, el uso de bucles y la gestión de estructuras de selección.

El programa se ejecuta en la terminal y comienza solicitando los datos de dos dragones (nombre y edad). El sistema incluye un control de errores (try-except) para asegurar que la edad sea un número entero. Según esta edad, el dragón se clasifica automáticamente como Joven, Adulto o Anciano, lo que determinará sus estadísticas base.

Antes del combate, el código ejecuta una fase de entrenamiento: un bucle for simula 3 días de preparación donde las estadísticas de fuerza y resistencia aumentan según la categoría del dragón. Finalmente, se entra en un bucle while que gestiona la batalla por turnos, calculando el daño en tiempo real mediante una fórmula propia hasta que uno de los oponentes (o ambos) cae derrotado.

Características Principales

  • Validación de Datos: Uso de bloques try-except para evitar fallos si el usuario introduce texto en lugar de números.
  • Modularidad con Funciones: El cálculo de la fuerza inicial (`calculaFuerzaBase`) y del daño (`turnoDeAtaque`) se ha separado en funciones para limpiar el flujo principal.
  • Lógica de Entrenamiento: Un bucle iterativo (`for`) modifica las variables de estado antes del combate, premiando a los dragones jóvenes con un crecimiento más rápido.
  • Sistema de Combate: Bucle condicional (`while`) que enfrenta a las entidades turno a turno, con condiciones finales para determinar la victoria o el empate técnico.
