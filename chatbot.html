Chatbot IA Local

Integración de Qwen Coder con PHP y Ollama

Sobre el Proyecto

Este asistente virtual nació como una práctica reciente en clase para explorar la integración de modelos de lenguaje (LLMs) directamente en aplicaciones web. El objetivo era crear una alternativa privada a ChatGPT, ejecutando el modelo localmente sin depender de APIs externas de pago.

Para lograrlo, utilicé Ollama como servidor local de inferencia, corriendo el modelo qwen2.5-coder:7b (optimizado para programación y lógica). El backend está construido íntegramente en PHP, encargándose de gestionar las sesiones del usuario y actuar como puente (mediante peticiones cURL) entre la interfaz web y el motor de inteligencia artificial.

Características Técnicas

  • Consumo de API REST: Uso de curl en PHP para empaquetar los prompts en JSON y procesar la respuesta del servidor Ollama.
  • Persistencia de Contexto: Implementación de $_SESSION para almacenar el historial de preguntas y respuestas, manteniendo el flujo de la conversación activo incluso al recargar la página.
  • Ingeniería de Prompts (Básica): Manipulación de la cadena de texto antes de enviarla al modelo para forzar respuestas concisas ("Resume tu respuesta en una única frase").
  • Diseño UI/UX (CSS Puro): Interfaz inspirada en WhatsApp o iMessage, con sistema de burbujas dinámicas (flexbox) creadas con CSS sin librerías externas.
  • Patrón PRG (Post/Redirect/Get): Uso de header("Location: ...") tras procesar el formulario para evitar el molesto reenvío de datos al refrescar la web.

El Motor de Conexión

Este es el fragmento clave del backend en PHP encargado de comunicarse con la API de Ollama:

// Preparamos los datos para Ollama
$data = [
  "model" => $MODEL,
  "prompt" => $prompt,
  "stream" => false
];

// Iniciamos cURL hacia http://localhost:11434/api/generate
$ch = curl_init($OLLAMA_URL);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, ["Content-Type: application/json"]);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));

$response = curl_exec($ch);
curl_close($ch);
