Calculadora de Matrices

Herramienta Matemática CLI v3.1.0

Sobre el Proyecto

La Calculadora de Matrices (CdM) es un proyecto que nació durante mis estudios de 1º y 2º de Bachillerato, en la asignatura optativa de Programación, Redes y Sistemas Informáticos (PRSI). Lo que comenzó como un ejercicio básico evolucionó hasta convertirse en la versión v3.1.0, una herramienta robusta capaz de resolver problemas complejos de álgebra lineal.

El objetivo principal fue implementar algoritmos matemáticos desde cero ("vanilla Python"), sin depender de librerías externas como NumPy. Esto me permitió comprender a fondo la lógica detrás de operaciones como la inversión de matrices (usando adjuntos y determinantes) o la multiplicación matricial, traduciendo fórmulas matemáticas puras a estructuras de control como bucles anidados.

Funcionalidades Matemáticas

  • Operaciones Básicas: Suma y Resta de matrices de cualquier dimensión (previa validación de igualdad dimensional).
  • Multiplicación Matricial: Algoritmo de triple bucle anidado que verifica la compatibilidad (columnas A = filas B).
  • Determinantes: Cálculo exacto para matrices cuadradas de orden 2 y 3 mediante la Regla de Sarrus.
  • Matriz Inversa: Implementación completa del método de la matriz adjunta y traspuesta para órdenes 2x2 y 3x3.
  • Interfaz de Menú: Sistema de navegación por consola que permite realizar múltiples operaciones consecutivas sin reiniciar el programa.

Terminal Interactiva Web

He adaptado el código Python original para que se ejecute directamente en tu navegador usando WebAssembly.

Puedes probar todas las operaciones matemáticas aquí mismo:
